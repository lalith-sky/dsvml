# DSVML – Machine Learning Project

This repository contains the notebook and code files used for the DSVML machine learning project.

## Project Overview
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training & evaluation
- Visualization of results

## Directory Structure
Dsv.ipynb               – Main notebook
requirements.txt        – All dependencies
.gitignore              – Ignored files

## Install dependencies:
pip install -r requirements.txt

## Run the notebook:
Open Dsv.ipynb in Jupyter or Google Colab.
